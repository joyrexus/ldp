README

Kristi (on behalf of Susan Levine) is requesting a summary report of child gesture counts based on the LRB column from sessions 1 to 5 and 8 of the home visits.

Reports

The first report (summary-1.xls) provides summary counts of each of the LRB codes found as values in the LRB column.

The second report (summary-2.xls) provides summary counts of each of the LRB codes and their co-occurence with the gesture type codes found in the G_TYPE column.

The third report (summary-3.xls) provides summary counts of each of the LRB codes and their co-occurence with the gesture-speech relation codes found in the GS_REL column.

LRB Codes

The valid LRB codes are listed below. See the transcript column specification for an overview of the valid codes for the other gesture columns.

CODE  MEANING
L     Left hand
R     Right hand
B     Both hands
H     Head
LF    Left foot
RF    Right foot
W     Whole body

Columns

  • PROJ - project ID (2 or 3)
  • SUBJ - subject ID
  • SESS - session ID
