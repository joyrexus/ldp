3.12. DPP - Disjoint Prepositional Phrase Count
===============================================


.. _dpp-column-description:

3.12.1. Description
-------------------

Number of :term:`disjoint prepositional phrase`\ s in transcribed utterance


.. _dpp-column-format:

3.12.2. Format
--------------

.. productionlist::
    entry: <A non-negative integer>


.. _dpp-column-values:

3.12.3. Values
--------------

Any non-negative integer
