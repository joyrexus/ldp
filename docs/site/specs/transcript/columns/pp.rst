3.11. PP - Prepositional Phrase Count
=====================================


.. _pp-column-description:

3.11.1. Description

Number of Prepositional Phrases in transcribed utterance


.. _pp-column-format:

3.11.2. Format
--------------

.. productionlist::
   entry: <A non-negative integer>


.. _pp-column-values:

3.11.3. Values
--------------

Any non-negative integer

3.11.4. Vertical Dependencies
